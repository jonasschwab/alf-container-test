# ALF + pyALF Jupyter Container

This image includes the source code of [ALF](https://git.physik.uni-wuerzburg.de/ALF/ALF) and [pyALF](https://git.physik.uni-wuerzburg.de/ALF/pyALF), has their prerequisites installed and HDF5 precompiled. Since it is derived from jupyter/minimal-notebook, [this documentation](https://jupyter-docker-stacks.readthedocs.io) applies.
